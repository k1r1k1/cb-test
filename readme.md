## Create .env:
>```
>APP_SECRET=<appsecret>
>DB_HOST=<localhost>
>DB_NAME=<dbname>
>DB_USER=<username>
>DB_PASSWORD=<password>
>```

## End-point: http://localhost:3000/register
### Method: POST
>```
>http://localhost:3000/register
>```
### Body (**raw**)

```json
{ 
    "username": "username@mail.com",
    "password": "password"
}
```
